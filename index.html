Introduction

Currently Planning Center offers two different APIs that you can work with - a legacy API for PCO Services, and an all new API for your PCO account. Currently the new PCO API only has data from PCO People, but we are in the process of adding some of our other apps to it including Services.

Getting Started with the PCO API

The best place to get started with the PCO API is at the PCO API Documentation. There is also a Ruby gem wrapper around our API.

Getting Started with the PCO Services API

Documentation, examples, and getting started guides are available at this repo.

Support

If you have any questions or run into any hiccups, please open open an issue on our developer repo.

PCO Developers

Planning Center Online is a development driven company - over half of our staff are either developers or designers (designers who code!), our owner spends most of his time developing, and our developers maintain several popular open source projects and have contributed to many others including Rails and RSpec.

If you're a like minded developer, take a look at our careers page.
